2.3 SERVICE-WORKER.JS (3.4 KB)
─────────────────────────────────────────────────────────────────────────────

O que é:
- Script que roda em background
- Funciona offline
- Gerencia cache de assets

Versão de Cache:
- CACHE_NAME: 'organistuda-v3.5.7'
- Muda a cada atualização
- Invalida cache antigo automaticamente

Estratégia de Cache:

1. Cache-First (para assets estáticos):
   ├─ Tenta servir do cache primeiro
   ├─ Se não encontrar, faz fetch da rede
   ├─ Atualiza cache com nova versão
   └─ Aplicado a: /assets/, .png, .jpg, .svg, .css, .woff, .ttf

2. Stale-While-Revalidate (para HTML/JS):
   ├─ Serve versão em cache imediatamente
   ├─ Faz fetch em background
   ├─ Atualiza cache com nova versão
   └─ Próxima visita terá versão atualizada

Eventos Gerenciados:

Install:
- Cachear todos os assets estáticos
- skipWaiting(): Ativar imediatamente

Activate:
- Remover caches antigos
- claim(): Assumir controle de clientes

Fetch:
- Interceptar requisições HTTP
- Aplicar estratégia de cache apropriada
- Falhar graciosamente offline

Assets Cacheados:
- /index.html
- /manifest.json
- /icon-48.png até /icon-512.png
- /assets/index.css
- /assets/index.js

Comportamento Offline:
- Assets estáticos: Servem do cache
- HTML/JS: Servem do cache com atualização em background
- Requisições externas: Falham graciosamente
- Retorna página offline se necessário

11. COMO USAR O BUILD
================================================================================

11.1 DEPLOY EM SERVIDOR
─────────────────────────────────────────────────────────────────────────────

Passos:

1. Copiar conteúdo de dist/ para servidor web
   ```
   scp -r dist/* user@server:/var/www/organistuda/
   ```

2. Configurar servidor web (nginx):
   ```
   server {
     listen 443 ssl http2;
     server_name organistuda.com;
     
     root /var/www/organistuda;
     index index.html;
     
     # Gzip compression
     gzip on;
     gzip_types text/plain text/css application/json application/javascript;
     gzip_level 9;
     
     # Cache headers
     location ~* \.(js|css|png|jpg|svg|woff|woff2)$ {
       expires 1y;
       add_header Cache-Control "public, immutable";
     }
     
     # Service worker (não cachear)
     location = /service-worker.js {
       expires -1;
       add_header Cache-Control "no-cache, no-store, must-revalidate";
     }
     
     # HTML (não cachear)
     location = /index.html {
       expires -1;
       add_header Cache-Control "no-cache, no-store, must-revalidate";
     }
     
     # SPA routing
     location / {
       try_files $uri $uri/ /index.html;
     }
   }
   ```

3. Configurar HTTPS (Let's Encrypt):
   ```
   certbot certonly --webroot -w /var/www/organistuda -d organistuda.com
   ```

4. Testar:
   - Abrir https://organistuda.com
   - Verificar console (F12)
   - Testar offline (DevTools → Network → Offline)
