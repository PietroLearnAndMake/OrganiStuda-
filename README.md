# 🚀 OrganiStuda: Seu Companheiro de Estudos para o ENEM

![OrganiStuda Logo](public/icon-512.png)

Bem-vindo ao **OrganiStuda**, o aplicativo definitivo para otimizar sua preparação para o ENEM! Este é um projeto com **fins estritamente educativos**, desenvolvido por **Pietro**, um estudante dedicado que busca facilitar a rotina de estudos através da tecnologia e gamificação.

O OrganiStuda transforma a preparação para o vestibular em uma jornada envolvente, produtiva e recompensadora, focada no sucesso de cada estudante.

## ✨ Funcionalidades Principais

- **Banco de Questões Extenso:** Acesse um vasto banco de dados com milhares de questões do ENEM e de simulados renomados, categorizadas por matéria e origem.
- **Cronograma de Estudos Inteligente:** Organize seu tempo de estudo por matérias (Matemática, Linguagens, Humanas, Natureza) com um cronograma intuitivo.
- **Gamificação Avançada:**
  - **Sistema de Conquistas:** Desbloqueie mais de 25 medalhas exclusivas (Comum, Raro, Épico, Lendário) ao responder questões, manter sequências de acertos e concluir tópicos.
  - **XP e Níveis:** Ganhe pontos de experiência (XP) e suba de nível a cada avanço, transformando seu progresso em uma jornada recompensadora.
  - **Curva de Aprendizado Visual:** Acompanhe sua evolução e taxa de acertos ao longo do tempo com gráficos intuitivos, identificando pontos fortes e áreas de melhoria.
  - **Bônus de Sequência:** Mantenha o foco e seja recompensado com XP extra por sequências de acertos consecutivos.
- **Ferramentas de Produtividade Integradas:**
  - **Pomodoro Timer:** Gerencie seu tempo de estudo com a técnica Pomodoro, alternando entre foco intenso e pausas estratégicas.
  - **Lista de Tarefas:** Organize suas atividades diárias e acompanhe seu progresso.
- **Experiência Imersiva:** O aplicativo é otimizado para tela cheia (modo imersivo) em dispositivos Android, proporcionando um ambiente de estudo sem distrações.
- **Interface Moderna e Intuitiva:** Desfrute de um design limpo, responsivo e com opções de modo claro/escuro para uma experiência visual agradável.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 19, TypeScript, Vite
- **Estilização:** Tailwind CSS 4
- **Animações:** Motion
- **Ícones:** Lucide React
- **Mobile:** Capacitor (para compilação Android)
- **Automação:** GitHub Actions (para builds e releases automáticas)

## 🚀 Como Instalar (Android)

1.  **Baixe o APK:** Acesse a [página de Releases](https://github.com/PietroLearnAndMake/Organizador-e-Banco-de-quest-es-para-Enem/releases) do projeto e faça o download da versão mais recente do `OrganiStuda.apk`.
2.  **Permitir Instalação de Fontes Desconhecidas:** No seu dispositivo Android, vá em `Configurações > Segurança` (ou `Aplicativos e notificações > Acesso especial a apps > Instalar apps desconhecidos`) e permita a instalação de aplicativos de fontes desconhecidas.
3.  **Instale o APK:** Abra o arquivo APK baixado e siga as instruções para instalar o OrganiStuda.
4.  **Comece a Estudar!**

## 💻 Desenvolvimento

Para rodar o projeto localmente:

```bash
git clone https://github.com/PietroLearnAndMake/Organizador-e-Banco-de-quest-es-para-Enem.git
cd Organizador-e-Banco-de-quest-es-para-Enem
npm install
npm run dev
```

Para buildar para Android (requer Android SDK e Java 17):

```bash
npm run build
npx cap sync android
cd android && ./gradlew assembleDebug
```
